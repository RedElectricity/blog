## 阅读前的小提示

由于德语的词性在我看来就是依托石, 因此本文章只能涵盖大部分词的规律, 无法覆盖所有词

## 词性

### Der

#### 男性的职业/身份

| 后缀               | 例子           |
| ------------------ | -------------- |
| -er                | der Lehrer     |
| -or                | der Autor      |
| -eur               | der Friseur    |
| -and (阳性弱变化)  | der Diplomand  |
| -ist (阳性弱变化)  | der Journalist |
| -ent (阳性弱变化)  | der Student    |
| -ant (阳性弱变化)  | der Praktikant |
| -naut (阳性弱变化) | der Astronaut  |
